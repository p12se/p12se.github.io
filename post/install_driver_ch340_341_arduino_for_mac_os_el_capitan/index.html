Установка драйверов CH340, CH341 для китайстойкой arduino в mac os el capitan

Для того, что бы подключить китайскую arduino mega, пришлось мне изрядно повозиться.

Проблема заключалась в том, что в устройство не определялось на моей системе El Capitan. Простая установка драйверов под чипы CH340G, CH340, CH341 не давала необходимого результата.

Для решения этой проблемы необходимо проделать следующие действия:
* Перезагружаем свою систему;
* Во время загрузки нажимаем CMD+R после загрузки попадаем в режим восстановления;
* Запускаем терминал и вводим команду csrutil enable --without kext;
* перезагружаем.

После того как загрузилась система необходимо скачать драйвера http://www.wch.cn/download/CH341SER_MAC_ZIP.html

Устанавливайте их, затем в Arduino.app переходим в Инструменты -> Порт
